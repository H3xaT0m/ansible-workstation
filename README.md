# Prerequisites

* Installed Manjaro
  * Manjaro i3
    * tz=Europe/Berlin
    * keytable=us
    * lang=de_DE
    * driver=nonfree
* Installed Ansible
* Installed Git

# If you're brave, you can use the `initial_run.sh` script as following:
```bash
curl -s https://raw.githubusercontent.com/H3xaT0m/ansible-workstation/master/initial_run.sh
```
